# Lección 02 - Primer contacto con CodePen

## Objetivo de la lección

Iniciar el proyecto principal del curso creando el primer contenido de una página web personal mediante HTML.

Al finalizar la lección, el estudiante debe poder crear un título principal y varios párrafos usando etiquetas HTML básicas.

## Conceptos clave

- Editor online.
- Etiqueta HTML.
- Título principal con `h1`.
- Párrafo con `p`.
- Etiqueta de apertura y etiqueta de cierre.
- Vista previa del navegador.

## Código que se desarrollará

### HTML

```html
<h1>Mi Perfil Web</h1>

<p>Hola, esta es mi primera página web.</p>

<p>
  Estoy aprendiendo HTML para crear contenido
  y CSS para mejorar el diseño de mis páginas.
</p>

<p>
  Mi objetivo es construir una página sencilla para presentarme
  y compartir algunos intereses.
</p>
```

```css
/* Los estilos CSS se trabajarán en próximas lecciones */
```

## Explicación sugerida

La lección inicia recordando que HTML sirve para crear el contenido de una página web.

Luego se crea un título principal usando la etiqueta `h1`.

Después se agregan párrafos usando la etiqueta `p`.

Se explica que las etiquetas suelen tener apertura y cierre, por ejemplo `<h1>` y `</h1>`, o `<p>` y `</p>`.

También se muestra que el editor online permite ver los cambios inmediatamente en la vista previa.

## Práctica para el estudiante

Cambiar el texto del primer párrafo y agregar un nuevo párrafo contando algo que le gustaría aprender sobre páginas web.
