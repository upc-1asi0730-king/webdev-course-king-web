# Lección 03 - Estructura básica de HTML

## Objetivo de la lección

Organizar el contenido inicial de una página web personal usando etiquetas básicas de estructura HTML.

Al finalizar la lección, el estudiante debe poder usar `header`, `main` y `section` para ordenar mejor el contenido de una página.

## Conceptos clave

- Estructura HTML.
- Etiqueta `header`.
- Etiqueta `main`.
- Etiqueta `section`.
- Subtítulo con `h2`.
- Organización del contenido.
- Etiquetas de apertura y cierre.

## Código que se desarrollará

### HTML

```html
<header>
  <h1>Mi Perfil Web</h1>
  <p>Una página para presentarme al mundo.</p>
</header>

<main>
  <section>
    <h2>Sobre mí</h2>

    <p>
      Hola, esta es mi primera página web.
    </p>

    <p>
      Estoy aprendiendo HTML para crear contenido
      y CSS para mejorar el diseño de mis páginas.
    </p>

    <p>
      Mi objetivo es construir una página sencilla para presentarme
      y compartir algunos intereses.
    </p>
  </section>
</main>
```

## Explicación sugerida

La lección continúa desde el avance anterior, donde ya existe un título principal y varios párrafos.

Primero se explica que el código funciona, pero puede organizarse mejor.

Luego se presenta `header` como la parte inicial de la página.

Después se presenta `main` como el contenido principal.

Finalmente, se usa `section` para agrupar contenido relacionado y se agrega un subtítulo con `h2`.

## Práctica para el estudiante

Cambiar el texto del subtítulo “Sobre mí” y agregar un nuevo párrafo dentro de la sección.
