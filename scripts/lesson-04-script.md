# Lección 04 - Textos y listas en HTML

## Objetivo de la lección

Agregar más contenido a una página web personal usando listas en HTML.

Al finalizar la lección, el estudiante debe poder crear listas sin orden y listas ordenadas usando las etiquetas `ul`, `ol` y `li`.

## Conceptos clave

- Texto en HTML.
- Subtítulo con `h2`.
- Párrafo con `p`.
- Lista sin orden con `ul`.
- Lista ordenada con `ol`.
- Elemento de lista con `li`.
- Organización de contenido con `section`.

## Código que se desarrollará

### HTML

```html
<header>
  <h1>Mi Perfil Web</h1>
  <p>Una página para presentarme al mundo.</p>
</header>

<main>
  <section>
    <h2>Sobre mí</h2>

    <p>
      Hola, esta es mi primera página web.
    </p>

    <p>
      Estoy aprendiendo HTML para crear contenido
      y CSS para mejorar el diseño de mis páginas.
    </p>

    <p>
      Mi objetivo es construir una página sencilla para presentarme
      y compartir algunos intereses.
    </p>
  </section>

  <section>
    <h2>Mis hobbies</h2>
    <p>Estas son algunas actividades que disfruto:</p>

    <ul>
      <li>Escuchar música</li>
      <li>Jugar fútbol</li>
      <li>Dibujar</li>
      <li>Aprender sobre tecnología</li>
    </ul>
  </section>

  <section>
    <h2>Mis metas</h2>
    <p>Estos son algunos objetivos que quiero lograr:</p>

    <ol>
      <li>Aprender HTML</li>
      <li>Diseñar con CSS</li>
      <li>Crear mi primera página completa</li>
    </ol>
  </section>
</main>
```

## Explicación sugerida

La lección continúa desde la estructura básica creada anteriormente.

Primero se agrega una nueva sección para mostrar hobbies.

Luego se explica que una lista sin orden se crea con `ul` y que cada elemento de la lista se escribe con `li`.

Después se agrega una sección de metas.

Finalmente, se explica que una lista ordenada se crea con `ol`, y que el navegador muestra sus elementos con números.

## Práctica para el estudiante

Cambiar los hobbies por actividades propias y agregar una meta adicional dentro de la lista ordenada.
