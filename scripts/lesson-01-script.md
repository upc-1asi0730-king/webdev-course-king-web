# Lección 01 - ¿Qué es una página web?

## Objetivo de la lección

Explicar qué es una página web y presentar la diferencia básica entre HTML y CSS.

Al finalizar la lección, el estudiante debe entender que HTML se usa para crear contenido y CSS se usa para modificar la apariencia visual de una página web.

## Conceptos clave

- Página web.
- Navegador web.
- HTML como contenido.
- CSS como estilo.
- Editor online.

## Código que se desarrollará

### HTML

```html
<h1>Mi primera página web</h1>

<p>Estoy aprendiendo cómo se construyen los sitios web.</p>
```

### CSS

```css
body {
  background-color: #f0f6ff;
  font-family: Arial, sans-serif;
}

h1 {
  color: royalblue;
}

p {
  font-size: 18px;
}
```

## Explicación sugerida

La lección inicia explicando que una página web es un documento que se abre desde un navegador.

Luego se presenta HTML como la tecnología que permite crear contenido, por ejemplo títulos y párrafos.

Después se presenta CSS como la tecnología que permite modificar la apariencia visual del contenido, por ejemplo colores, tamaños, fuentes y fondo.

Se usa una comparación sencilla: HTML es como la estructura de una casa y CSS es como la pintura y la decoración.

## Práctica para el estudiante

Cambiar el texto del título y probar otro color para el `h1`, por ejemplo `green`, `purple` u `orange`.
